# 실시간 이상탐지 서비스

온도, 진동, 소리 데이터를 활용하여 실시간으로 이상을 탐지하여 관리자에게
경고메세지를 보내 설비를 예지보전하는 프로젝트입니다. 
## 프로젝트 내용
| 항목 | 내용 |
|------|------|
|**프로젝트명** | 실시간 이상탐지 서비스 |
| **개발기간** | 2024.09 ~ 2024.10 |
| **주요기술** | python, LSTM, AUTOENCODER, SVM, Telegram API|
| **데이터 규모** | 온도 : 약 6300장, 진동 : 약 1200만장, 소리 : 약 20000만개 |
| **성과** | 정확도 91%, 앱 연동 성공 |
