# 반려동물 조기질환 AI 애플리케이션

반려동물들의 이미지를 분석하여 질병 여부와 질병에 관한 정보를 알려주는 애플리케이션입니다.
AI 모델, Backend 서버, Frontend 앱으로 구성된 풀스택 프로젝트입니다.
